Herramienta de cotización para talleres de maquinados industriales. Configura rangos de precios, personaliza colores, aplica presets y guarda tu configuración en el navegador.

## ✨ Live Demo

👉 **[Ver demo en vivo](https://abrahamramoskd.github.io/maquinados_industriales/cifras.html)**


## 🔧 Funcionalidades

- 💰 **Rangos de precios configurables** — ajusta los valores por tipo de maquinado
- 🎨 **Editor de paleta de colores** — personaliza la apariencia de la herramienta
- 📋 **Presets listos** — configuraciones de arranque para casos comunes
- 💾 **Guardado en localStorage** — tu configuración persiste entre sesiones

## 🛠️ Tecnologías

- HTML5 semántico
- CSS puro
- Vanilla JS + localStorage API

## 📁 Estructura

```
maquinados_industriales/
├── cifras.html              ← Aplicación principal
└── maquinados_dashboard.png ← Preview del diseño
```

## 🚀 Uso

Abre `cifras.html` en tu navegador. Ajusta los rangos de precios según el tipo de maquinado y guarda tu configuración para usarla en futuras sesiones.
